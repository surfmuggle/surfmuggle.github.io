## Some Tools

- [Forgejo](https://forgejo.org/) is a self-hosted lightweight software forge.
- [pocketbase](https://pocketbase.io/) provides a single binary backed and cas be used as a go framework
- [plane](https://github.com/makeplane/plane) an issue tracker and project management tool written in python + typescript using django
- [readeck](https://codeberg.org/readeck/readeck) a web application that lets you saves content of web pages https://readeck.org/ that also provides a browser extension (a bookmark manager and a read later tool).
